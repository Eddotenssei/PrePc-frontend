# PrePC - Frontend

**PrePC** рЃљрЃарЃўрЃА рЃЋрЃћрЃЉ-рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃА рЃљрЃФрЃџрЃћрЃЋрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃЮрЃюрЃџрЃљрЃўрЃю рЃарЃћрЃЪрЃўрЃЏрЃерЃў рЃљрЃљрЃгрЃДрЃЮрЃю рЃАрЃљрЃЎрЃБрЃЌрЃљрЃарЃў рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃў рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃљрЃарЃЕрЃћрЃЋрЃўрЃЌ. рЃћрЃА рЃљрЃарЃўрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА Front-end рЃюрЃљрЃгрЃўрЃџрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃА рЃЊрЃљ API-рЃАрЃЌрЃљрЃю рЃЎрЃЮрЃЏрЃБрЃюрЃўрЃЎрЃљрЃфрЃўрЃљрЃА рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА.

---

## ­Ъїљ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃљрЃдрЃгрЃћрЃарЃљ

PrePC рЃЏрЃўрЃќрЃюрЃљрЃЊ рЃўрЃАрЃљрЃ«рЃљрЃЋрЃА рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃўрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃўрЃА рЃњрЃљрЃЏрЃљрЃарЃбрЃўрЃЋрЃћрЃЉрЃљрЃА рЃЋрЃћрЃЉ-рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃљрЃќрЃћ. рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃА рЃерЃћрЃБрЃФрЃџрЃўрЃљрЃЌ рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃњрЃљрЃЊрЃљрЃарЃЕрЃћрЃЋрЃљ, рЃЎрЃљрЃџрЃљрЃЌрЃљрЃерЃў рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ рЃЊрЃљ рЃАрЃљрЃЉрЃЮрЃџрЃЮрЃЮ PC-рЃА рЃљрЃгрЃДрЃЮрЃЉрЃљ рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЊрЃљ рЃўрЃюрЃбрЃБрЃўрЃфрЃўрЃБрЃарЃў рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ.

---

## ­Ъњ╗ рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃБрЃарЃў рЃАрЃбрЃћрЃЎрЃў

- **HTML** РђЊ рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ  
- **CSS** РђЊ рЃАрЃбрЃўрЃџрЃў  
- **JavaScript** РђЊ рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃў рЃЊрЃљ API-рЃЌрЃљрЃю рЃЎрЃЮрЃЏрЃБрЃюрЃўрЃЎрЃљрЃфрЃўрЃљ  
- **Fetch API** РђЊ Backend-рЃЌрЃљрЃю рЃЎрЃљрЃЋрЃерЃўрЃарЃўрЃАрЃЌрЃЋрЃўрЃА  
- **Responsive Layout** РђЊ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃљрЃќрЃћ рЃЏрЃЮрЃарЃњрЃћрЃЉрЃБрЃџрЃў рЃЊрЃўрЃќрЃљрЃўрЃюрЃў
